# Sa-Token 权限管理系统

基于 Spring Boot + Vue 3 + Sa-Token 的前后端分离权限管理系统。

## 功能特点

- 基于 Sa-Token 的权限认证
- 前后端分离架构
- 用户管理（增删改查）
- 角色权限管理
- 完整的权限控制体系
- 使用 Element Plus 构建现代化 UI

## 技术栈

### 后端
- Spring Boot 3.x
- MyBatis Plus
- Sa-Token 权限框架
- MySQL 数据库

### 前端
- Vue 3
- Vite
- Element Plus
- Pinia 状态管理
- Axios HTTP 客户端

## 快速开始

### 后端启动
```bash
mvn spring-boot:run
```

### 前端启动
```bash
cd satoken
npm install
npm run dev
```
